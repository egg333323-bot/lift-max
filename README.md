# Lift Max - 1RM Performance Calculator

A beautiful, responsive single-page web app for estimating your one-rep max (1RM) using the Epley formula.

## Features
- Clean dark-mode UI with gold/red accents
- 1RM calculation: `weight × (1 + reps / 30)`
- Optional bodyweight multiplier
- Percentage-based training table (100% down to 40%)
- Fully responsive & mobile-friendly
- No backend required

## How to Run Locally
1. Download or clone this repository
2. Open `index.html` in your browser

## Deploy to GitHub Pages
1. Create a new GitHub repository
2. Upload these files:
   - `index.html`
   - `README.md`
3. Go to **Settings → Pages**
4. Set source to `main` branch → Save
5. Your app will be live at `https://yourusername.github.io/repo-name/`
